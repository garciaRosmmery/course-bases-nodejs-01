# Tabla de Multiplicar

Aplicación de consola en Nodejs

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base de la tabla de multiplicar            [número] [requerido]
  -l, --list     Muestra la tabla por consola        [booleano] [defecto: false]
  -t, --to       Indica hasta dónde realiza la tabla      [número] [defecto: 10]
  ```
